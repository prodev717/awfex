API Documentation

Complete reference for the AWFEX API endpoints.

GET /workflow

Returns a list of all saved workflow names.

POST /workflow

Saves or updates a workflow definition.

{
  "name": "mathFlow",
  "workflow": { "mul": [{ "add": [2, 3] }, 4] }
}
GET /workflow/:name

Retrieve a workflow definition by its name.

DELETE /workflow/:name

Permanently delete a workflow.

GET /run/:name

Execute a stored workflow. Supports query params.

GET /run/mathFlow?x=10&y=20
POST /run

Execute a workflow immediately. Supports query params.

POST /run?x=5
{
  "add": ["$query:x", 3]
}
GET /functions

Returns a list of all available function names.

GET /descriptions

Returns a map of functions and their descriptions/tooltips.

Special Values

$query:key

Injects values from URL query parameters.

$env:KEY

Injects server-side environment variables.
